Lakbay — Transportation Search Platform

Lakbay is a personal project I architected and built from the ground up to help people discover transportation routes across the Philippines. The platform connects origins, destinations, transit hubs, operators, and multi-hop travel routes through an intelligent search engine.

I designed the application architecture, relational data model, and technical foundation, then built scalable Spring Boot APIs with optimized database models for transportation search, content management, and geospatial data processing. Google Maps APIs were integrated to automate transportation data validation, enrichment, and administrative workflows.

  • End-to-end ownership of system architecture, infrastructure, deployments, and product roadmap
  • Scalable backend APIs for search, content management, and geospatial processing
  • Multi-hop route discovery across transit hubs and operators nationwide
  • Google Maps integration for data quality and operational efficiency
